# music-metadata
Music Metadata is a lightweight search tool built in a 2-day Scrum challenge during an SAP internship. It allows users to find songs and retrieve key data like artist, album, lyrics, and streaming links. The project focuses on learning Streamlit, Python logic, and API integration while delivering a simple MVP.

# Download

Clone the repo
`git clone https://github.com/aeren1010/music-metadata.git`

Run with Streamlit
`streamlit run app.py`
